HTML Basics

HTML (HyperText Markup Language) is the standard markup language used for creating web pages and applications.

HTML defines the structure of a web page by using a series of elements and attributes.

Here are some basic HTML elements:

  • <html> - The root element of an HTML document
  • <head> - Contains meta information about the document
  • <body> - Contains the main content of the web page
  • <h1> to <h6> - Heading elements of different sizes
  • <p> - Paragraph element
<!DOCTYPE html> <html lang="en"> <head> <title>My Web Page</title> </head> <body> <h1>Welcome to my website!</h1> <p>This is a simple HTML page.</p> </body> </html>
CSS Fundamentals

CSS (Cascading Style Sheets) is a language used to style and format the layout of a web page.

CSS allows you to control the appearance of elements like fonts, colors, backgrounds, and layouts.

Here are some basic CSS properties:

  • color - Sets the text color of an element
  • font-family - Specifies the font family for an element
  • background-color - Sets the background color of an element
  • margin - Sets the margin around an element
  • padding - Sets the padding inside an element
body { font-family: Arial, sans-serif; margin: 0; padding: 20px; } h1 { color: blue; text-align: center; }
JavaScript Introduction

JavaScript is a high-level, interpreted programming language that adds interactivity to web pages.

JavaScript can be used to manipulate the content of a web page, respond to user events, and create dynamic web applications.

Here's a simple example of JavaScript:

function greetUser() { alert("Hello, world!"); } greetUser();

JavaScript also allows you to work with data structures like arrays and objects.

const myArray = [1, 2, 3]; console.log(myArray[0]); // Output: 1 const myObject = { name: "John", age: 30 }; console.log(myObject.name); // Output: John

Version Control

Version control systems (like Git) are used to track changes to code over time.

Git allows developers to collaborate on projects more effectively by enabling them to work on different branches and merge their changes.

Here are some key Git commands:

  • git init - Creates a new Git repository
  • git add - Stages changes for commit
  • git commit - Creates a snapshot of the current state of the repository
  • git push - Pushes changes to a remote repository

Git also provides features like branching, merging, and resolving conflicts.

git checkout -b new-feature // Create a new branch git merge main // Merge changes from the main branch
